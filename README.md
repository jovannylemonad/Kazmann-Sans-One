# Kazmann Sans One

Kazmann Sans One font is a narrow grotesque designed for titles, mobile applications and games. The main feature of the font is a big contrast between the height of uppercase and lowercase characters, and the accenders and descenders elements.

The font contains 416 characters. Supported languages: belarus, czech, dutch, english, estonian, finnish, french, german, italian, hungarian, latvian, lithuanian, norwegian, polish, russian, spanish, turkish, ukrainian.
